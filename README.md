# 레진 알리미 (Resin Timer) 
## 간편한 잔여 레진 표시기

남은 레진을 확인하느라 게임에 접속하기 귀찮으셨나요?

레진을 전부 회복하는 잔여 시간 하나만 알면 편하게 남은 레진, 현재 레진, 1개당 남은 시간 등 자동적으로 계산되어 표시됩니다

# 기능

- 한 눈에 보기 쉬운 잔여 레진 표시
- 총 남은 회복 시간
- 1개당 남은 회복 시간
- 간단한 설정
